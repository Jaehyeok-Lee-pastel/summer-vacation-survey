📋 1. API 서버 상태 확인

GET /api/status/

API 서버가 정상 작동하는지 확인합니다.

✅ 성공 응답 (200)

{ "status": "OK", "message": "여름휴가 설문조사 API 서버가 정상 작동 중입니다!", "version": "1.0.0" }

👤 2. 회원가입 API

POST /api/register/

새로운 사용자 계정을 생성합니다.

📤 요청 데이터

필드 타입 필수 설명
name string ✅ 사용자 실명 (한글)
userid string ✅ 로그인용 아이디 (영문+숫자)
password string ✅ 비밀번호
password_confirm string ✅ 비밀번호 확인
age string ✅ 나이 (1-100 사이 숫자)
email string ✅ 이메일 주소

📝 요청 예시

{ "name": "홍길동", "userid": "honggildong123", "password": "mypassword123", "password_confirm": "mypassword123", "age": "25", "email": "hong@naver.com" }

✅ 성공 응답 (201)

{ "success": true, "message": "회원가입이 완료되었습니다.", "user": { "id": 7, "username": "honggildong123", "name": "홍길동", "email": "hong@naver.com", "age": 25 } }

❌ 실패 응답 (400)

{ "success": false, "message": "이미 사용중인 아이디입니다." }

⚠️ 가능한 에러 메시지

  • "이름을 입력해주세요."
  • "아이디를 입력해주세요."
  • "비밀번호를 입력해주세요."
  • "비밀번호가 일치하지 않습니다."
  • "올바른 나이를 입력해주세요."
  • "이미 사용중인 아이디입니다."
  • "이미 사용중인 이메일입니다."

📤 요청 데이터

필드 타입 필수 설명

{ "userid": "honggildong123", "password": "mypassword123" }

📝 4. 설문조사 제출 API

POST /api/survey/

설문조사 응답 데이터를 저장합니다.

📤 요청 데이터

필드 필수 설명 선택값
q1 ✅ 연령대 "10대", "20대", "30대", "40대", "50대", "60대 이상"
q2 ✅ 성별 "남성", "여성"
q3 ✅ 최근 여름휴가 활동 "해수욕, 물놀이", "등산, 캠핑", "문화생활", "도시 관광", "휴양·힐링", "맛집 투어", "친척·지인 방문", "기타"
q4 ✅ 휴가 장소 "국내", "해외"
q5 ✅ 주요 교통수단 "자동차", "버스", "기차", "항공편", "배", "도보"
q6 ✅ 휴가 기간 "1일", "2~3일", "4~6일", "7~15일", "15일 이상"
q7 ✅ 함께한 사람 "혼자", "가족", "친구", "연인", "직장 동료", "동호회", "기타"
q8 ✅ 지출 비용 "10만 원 이하", "10만~30만 원", "30만~50만 원", "50만~100만 원", "100만~200만 원", "200만 원 이상"
q9 ✅ 만족도 점수 1, 2, 3, 4, 5 (숫자)
q10 ✅ 다음 여름휴가 희망 "바다/섬에서 물놀이", "산·계곡에서 활동", "문화 체험", "도시 관광", "휴양·힐링", "맛집 탐방", "친척·지인 방문", "기타"

📝 요청 예시

{ "q1": "20대", "q2": "남성", "q3": "해수욕, 물놀이", "q4": "국내", "q5": "자동차", "q6": "2~3일", "q7": "가족", "q8": "30만~50만 원", "q9": 4, "q10": "바다/섬에서 물놀이" }

✅ 성공 응답 (200)

{ "success": true, "message": "설문조사가 저장되었습니다." }

🛠️ 5. 프론트엔드 연동 예시

👤 회원가입 연동 (register.html)

fetch("https://dt5857.pythonanywhere.com/api/register/", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ name: "홍길동", userid: "honggildong123", password: "mypassword123", password_confirm: "mypassword123", age: "25", email: "hong@naver.com" }) }) .then(res => res.json()) .then(data => { if (data.success) { alert("회원가입 완료!"); window.location.href = "login.html"; } else { alert(data.message); } });

🔐 로그인 연동 (login.html)

fetch("https://dt5857.pythonanywhere.com/api/login/", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ userid: "honggildong123", password: "mypassword123" }) }) .then(res => res.json()) .then(data => { if (data.success) { alert(data.message); // "홍길동님, 로그인 성공!" window.location.href = "survey.html"; } else { alert(data.message); } });

📝 설문조사 연동 (survey.html)

// 설문 데이터 수집 const surveyData = { q1: document.querySelector('input[name="q1"]:checked').value, q2: document.querySelector('input[name="q2"]:checked').value, q3: document.querySelector('input[name="q3"]:checked').value, // ... 나머지 필드들 q9: parseInt(document.querySelector('input[name="q9"]:checked').value), q10: document.querySelector('input[name="q10"]:checked').value }; fetch("https://dt5857.pythonanywhere.com/api/survey/", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify(surveyData) }) .then(res => res.json()) .then(data => { if (data.success) { alert("설문조사가 완료되었습니다!"); window.location.href = "result.html"; } else { alert(data.message); } });

⚠️ 6. 주의사항 및 테스트 방법

🚨 주의사항

CORS 설정: 프론트엔드 도메인이 Django settings.py의 CORS_ALLOWED_ORIGINS에 추가되어야 함

CSRF 토큰: 현재 @csrf_exempt 데코레이터로 비활성화되어 있음

인증: 설문조사 API에서 @login_required 데코레이터가 주석처리되어 있음

사용자 ID: 설문조사에서 현재 user = User.objects.get(id=1)로 임시 고정됨

🧪 Postman 테스트 예시

1단계: 회원가입 테스트

  • Method: POST
  • URL: https://dt5857.pythonanywhere.com/api/register/
  • Headers: Content-Type: application/json
  • Body: 위의 회원가입 JSON 데이터 복사

2단계: 로그인 테스트

  • Method: POST
  • URL: https://dt5857.pythonanywhere.com/api/login/
  • Headers: Content-Type: application/json
  • Body: 위의 로그인 JSON 데이터 복사

3단계: 설문조사 테스트

  • Method: POST
  • URL: https://dt5857.pythonanywhere.com/api/survey/
  • Headers: Content-Type: application/json
  • Body: 위의 설문조사 JSON 데이터 복사

🔍 브라우저 개발자 도구로 디버깅

// 디버깅용 코드 추가 fetch("https://dt5857.pythonanywhere.com/api/login/", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ userid, password }) }) .then(res => { console.log("응답 상태:", res.status); return res.json(); }) .then(data => { console.log("응답 데이터:", data); if (data.success) { // 성공 처리 } }) .catch(err => { console.error("에러:", err); });

📱 추천 테스트 순서

  1. Postman으로 API 먼저 테스트 → API가 정상 작동하는지 확인
  2. HTML + JavaScript로 실제 테스트 → 프론트엔드 연동 확인
  3. 브라우저 개발자 도구로 디버깅 → 문제 발생시 원인 파악
  4. 다른 컴퓨터에서 테스트 → 크로스 플랫폼 확인
